# dayflow-noai-minimal

Minimal macOS screen capture timeline (no AI).

## Run

```bash
deno task dev
```

Open [http://localhost:8787](http://localhost:8787).

## Check

```bash
deno task check
```

## Notes

- macOS only (`screencapture`, `lsappinfo`).
- Runtime artifacts are written to `data/` and ignored by git.
